# Notion-Quotes 🪷

A beautiful Notion widget that displays inspirational **Soka Global quotes** from Daisaku Ikeda, Nichiren Daishonin, Josei Toda, and Tsunesaburo Makiguchi.

## Live Widget

> Once deployed, the widget is available at:  
> `https://<your-github-username>.github.io/Notion-Quotes/`

## Embed in Notion

1. Copy the deployed URL above.
2. In Notion, type `/embed` and choose **Embed**.
3. Paste the URL and confirm.
4. Resize the embed block to your liking.

## Features

- 35+ curated Soka Global quotes
- Random starting quote on every page load
- Previous / Next navigation buttons
- Keyboard navigation (← / → arrow keys)
- Smooth fade transitions between quotes
- Dark, elegant design optimised for Notion embeds

## Quote Sources

- **Daisaku Ikeda** — Third president of Soka Gakkai International
- **Josei Toda** — Second president of Soka Gakkai
- **Tsunesaburo Makiguchi** — Founder of Soka Gakkai
- **Nichiren Daishonin** — 13th-century Buddhist monk whose teachings Soka Gakkai follows
